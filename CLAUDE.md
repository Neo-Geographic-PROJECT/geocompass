# GeoCompass v1.1 — 開発引き継ぎ

## プロジェクト概要
地理院タイルを使用した2地点の都市規模比較ツール。
メルカトル図法の歪みを動的に補正し、正確な距離感を視覚的に比較する。

## 確定仕様（Ver.1.0）
- 地理院ジオコーダによる座標・標高取得
- オーバーレイ方式（単一マップ＋半透明重畳）をデフォルト
- 左右分割モード（SPLIT VIEW）と切替可能
- 地点Bは初期値でAの中心にセンタリング
- 5枠プリセット＋ホームポジション（localStorage）
- ドロワー式パネル（右側）
- メルカトル歪み係数リアルタイム表示・自動補正

## 現在のファイル構成
map_compare_v1.1.html — 単体HTMLファイル（CSS/JS内包）

## 技術スタック
- Leaflet.js 1.9.4（CDN）
- 地理院タイル（std/pale/blank/photo/hillshademap）
- 地理院ジオコーダ API

- localStorage による状態永続化
